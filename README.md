# Escape-from-TuxKov
# Build
```
git clone --recursive https://github.com/Qemu-Gang/Escape-from-TuxKov
cd Escap*
mkdir build
cd build
cmake ..
make
```

# Run
- Make sure peeper is running

- `sudo ./TuxKovExternal`

# Notes
- janky between games, restart if it glitches out or stops working.
